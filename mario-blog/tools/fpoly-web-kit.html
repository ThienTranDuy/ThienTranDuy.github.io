3. Config our tool

3.1 Prerequisites

You need setup:

  • Visual Studio Code (https://code.visualstudio.com/)
  • Nodejs new version (https://nodejs.org/en/download/)
  • Python version 2.7.15 (https://www.python.org/downloads/release/python-2715/)
  • Yarn Package (https://yarnpkg.com/en/)

Then you need reset your computer!

3.2 Create repository

Create and variable about information of our tool

1. Create new folder and open it by Visual Studio Code

2. Open terminal by Ctrl + ~ then write this code:

yarn init

Answer the question, It like that:

File name package.json have been created when you done

3.3 Add loader, plugin, library webpack need to our tool

3.3.1 Wepack and them friends in mode developer dependencies (yarn add something --dev)
yarn add copy-webpack-plugin webpack webpack-cli webpack-dev-server webpack-merge webpack-notifier --dev
  • copy-webpack-plugin     : Use to copy file from folder `source` to folder `dist`
  • webpack                           : Add webpack to tools
  • webpack-dev-server       : Create server virtual in local
  • webpack-merge              : Merge webpack mode (common, develop, product)
  • webpack-notifier             : Alert infomation of user action
3.3.2 Add Sass loader to help translate Sass to CSS https://webpack.js.org/loaders/sass-loader/#root
yarn add css-loader mini-css-extract-plugin node-sass sass-loader style-loader --dev
3.3.3 Add Pug loader to help translate Pug to HTML https://github.com/pugjs/pug-loader
yarn add html-webpack-plugin pug pug-loader --dev
3.3.4 Add file loader to help translate file (img, jpg,..) in folder `src` to them in folder `dist`https://webpack.js.org/loaders/file-loader/#root
yarn add file-loader --dev
3.3.4 Add Library and Framework
  • Bootstrap https://getbootstrap.com/docs/4.3/getting-started/webpack/
  • Jquery https://jquery.com/download/#downloading-jquery-using-npm-or-yarn
  • popper.js https://github.com/FezVrasta/popper.js
  • Font Awesome https://fontawesome.com/how-to-use/on-the-web/setup/using-package-managers
yarn add @fortawesome/fontawesome-free bootstrap jquery popper.js
